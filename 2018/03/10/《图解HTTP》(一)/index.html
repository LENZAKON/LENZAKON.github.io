URL 和 URI

URI：由某个协议方案表示的资源的定位标识符

URI用字符串标识某一互联网资源，URL标识资源的地址（互联网上所处的位置）URL 是 URI 的子集

URI格式：

登录信息：指定用户名和密码，作为从服务端获取资源时必要的登录信息（可选）

服务器地址：域名，IPV4，IPV6

查询字符串：针对文件路径内的资源传入任意参数（可选）

片段标识符：标识以获取资源中的子资源（文档中的某个位置）
