对象转化中的toString和valueOf

toString()和valueOf()都是Object的方法。

在js中进行比较或者运算时，会把各种对象转化成Number,String,Boolean

valueOf：

Array 返回数组对象本身。

Boolean 布尔值。

Date 存储的时间是从 1970 年 1 月 1 日午夜开始计的毫秒数 UTC。

Function 函数本身。

Number 数字值。

Object 对象本身。这是默认情况。

String 字符串值。

Math 和 Error 对象没有 valueOf 方法。

toString：

String转化

在操作或者运算需要字符串的时候，会触发Object的String转化

转化步骤

  • 如果有toString方法，并且返回“原始类型”，返回toString的结果。
  • 如果toString方法不存在或者返回的不是“原始类型”，调用valueOf存在，且返回原始类型数据，返回valueOf的结果。
  • 报错

Number转化

方法参数需要Number的时候，如Math.sin(obj)等

对比的时候，如 obj == ‘abc’

运算的时候，如 obj + 123

转化步骤

  • 如果有valueOf方法，并且返回“原始类型”，返回valueOf的结果。
  • 如果valueOf方法不存在或者返回的不是“原始类型”，调用toString存在，且返回原始类型数据，返回toString的结果。
  • 报错

Boolean转化

if(obj) , while(obj)等

以下转化为false

  • undefined
  • null
  • -0 或 +0
  • NaN
  • ‘’

[] , {} 转化为true

例子：

parseInt 的作用是把一个字符串转化成数字
parseInt(0.00000004)是先把里面的参数转化成字符串"4e-8"，再把字符串转化成数字。

转化步骤

  • 忽略字符串前面的空格

  • 找到第一个非空格字符，如果第一个字符不是数字或者负号，返回NaN,即parseInt('')是NaN

  • 如果是，继续解析第二个字符，直到遇到非数字字符
